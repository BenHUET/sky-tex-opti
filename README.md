```
Usage:
  sky-tex-opti [options]

Options:
  --profile <profile> (REQUIRED)  Path to the MO2 profile to optimize.
  --output <output> (REQUIRED)    Path where to store the output.
  --diffuse <diffuse>             Target resolution for diffuse textures. [default: 2048]
  --normal <normal>               Target resolution for normal textures. [default: 1024]
  --parallax <parallax>           Target resolution for parallax textures. [default: 512]
  --material <material>           Target resolution for material textures. [default: 512]
  --other <other>                 Target resolution for other textures. [default: 512]
  --logging                       Write log files. [default: True]
  --version                       Show version information
  -?, -h, --help                  Show help and usage information
```
